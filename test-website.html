📋 كيفية تشغيل الموقع

1. تشغيل الخادم:

python backend/app.py

أو استخدم الملف المساعد:

python start_website.py

2. فتح الموقع:

http://localhost:5000

3. اختبار جميع الصفحات:

  • الصفحة الرئيسية: http://localhost:5000/home
  • المنيو: http://localhost:5000/menu
  • اطلب الآن: http://localhost:5000/order
  • موقعنا: http://localhost:5000/location
  • تواصل معنا: http://localhost:5000/contact
  • كود الخصم: http://localhost:5000/
