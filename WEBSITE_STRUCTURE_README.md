# 🚗 SUZZ Drive-Thru - الموقع الجديد الشامل

## 📋 نظرة عامة على الموقع

تم تطوير موقع SUZZ ليكون موقعاً شاملاً ومتكاملاً لأول وأسرع Drive-Thru في السويس. الموقع يركز على **السرعة** و**الراحة** مع تصميم أنيق وعصري.

## 🏗️ هيكل الموقع

### 📄 الصفحات الرئيسية:

#### 1. **الصفحة الرئيسية** (`home.html`)
- **الغرض**: الصفحة المرحبة التي تعرض قوة SUZZ
- **المميزات**:
  - فيديو Hero بـ auto-play للـ drive-thru (5-10 ثواني)
  - تأكيد على السرعة: "2-3 دقائق فقط"
  - تصميم يدل على السرعة والـ drive-thru
  - أزرار CTA للطلب المباشر وعرض المنيو
  - إحصائيات السرعة مع تأثيرات بصرية
- **العناصر البصرية**:
  - ألوان تدل على السرعة (برتقالي/أحمر للسرعة)
  - أيقونات البرق والسيارات
  - خلفية فيديو للـ drive-thru

#### 2. **صفحة المنيو** (`menu.html`)
- **الغرض**: عرض جميع المنتجات المتاحة
- **الأقسام**:
  - ☕ القهوة (إسبريسو، كابتشينو، لاتيه، آيس كوفي)
  - 🥤 المشروبات (عصائر طازجة، ليموناضة، موهيتو، ميلك شيك)
  - 🍽️ الطعام (قريباً جداً)
  - 🍪 الحلويات (كوكيز، كب كيك)
- **المميزات**:
  - نظام تبويب تفاعلي
  - كروت أنيقة لكل منتج
  - أسعار واضحة
  - تأكيد على السرعة "جاهز في 2-3 دقائق"

#### 3. **صفحة الموقع** (`location.html`)
- **الغرض**: معلومات الموقع وكيفية الوصول
- **المحتوى**:
  - خريطة الموقع (placeholder للتحديث لاحقاً)
  - معلومات العنوان وساعات العمل
  - تعليمات استخدام Drive-Thru (5 خطوات)
  - معلومات التواصل
- **المميزات**:
  - خطوات واضحة لاستخدام Drive-Thru
  - معلومات مواقف السيارات
  - أرقام التواصل مع روابط مباشرة

#### 4. **صفحة الطلب المسبق** (`order.html`)
- **الغرض**: حجز الطلبات مسبقاً
- **المميزات**:
  - نموذج طلب تفاعلي
  - اختيار المنتجات مع الكميات
  - حساب السعر الإجمالي
  - اختيار وقت الاستلام
  - ملاحظات خاصة
- **تحسينات UX**:
  - واجهة سهلة الاستخدام
  - تحديث فوري للأسعار
  - رسائل تأكيد وتحذير
  - تقدير وقت التحضير

#### 5. **صفحة كود الخصم** (`index.html` - محدث)
- **الغرض**: الحصول على كود خصم عبر رقم الهاتف
- **المميزات**:
  - نظام التحقق من رقم الهاتف
  - إنتاج QR Code للخصم
  - واجهة أنيقة متناسقة مع التصميم الجديد
  - نظام Dashboard للمستخدمين المسجلين

## 🎨 التصميم والألوان

### لوحة الألوان الرئيسية:
```css
:root {
    --primary: #1e293b;        /* أزرق داكن */
    --secondary: #334155;      /* رمادي أزرق */
    --accent: #059669;         /* أخضر SUZZ */
    --accent-light: #10b981;   /* أخضر فاتح */
    --accent-dark: #047857;    /* أخضر داكن */
    --speed: #ff6b35;          /* برتقالي للسرعة */
    --text: #f8fafc;           /* أبيض للنص */
    --background: #0f172a;     /* خلفية داكنة */
}
```

### المبادئ التصميمية:
1. **الوضوح**: نصوص واضحة وسهلة القراءة
2. **السرعة**: عناصر بصرية تدل على السرعة
3. **الأناقة**: تدرجات وتأثيرات بصرية حديثة
4. **سهولة الاستخدام**: navigation بسيط وواضح

## 📱 Navigation Bar

### المميزات:
- **Responsive**: يتكيف مع جميع أحجام الشاشات
- **Fixed Position**: يبقى مرئياً عند التمرير
- **Mobile Menu**: قائمة جانبية للهواتف المحمولة
- **Active States**: تمييز الصفحة النشطة
- **Smooth Animations**: انتقالات ناعمة

### الروابط:
- الرئيسية → `home.html`
- المنيو → `menu.html`
- الموقع → `location.html`
- الطلب المسبق → `order.html`
- كود الخصم → `index.html`

## 🚀 متطلبات الفيديو الرئيسي

- **الجودة**: 1080p أو أعلى

- **الحجم**: أقل من 10MB للتحميل السريع
